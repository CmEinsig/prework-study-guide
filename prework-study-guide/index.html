Prework Study Guide

HTML

  • HTML (Hypertext Mark Up)
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Basic building blocks of website.
  • Handles the content of the website.
  • One of three core languages

CSS

  • Cascading Style Sheets
  • Used to describe the presentation of the website
  • One of three core languages
  • Makes the website look pretty
  • Inline CSS: Applies to a specific HTML tag; permanent change, less flexible
  • Internal Style Sheet: Only applies to the page you're on
  • External CSS style sheet: Most common, easiest to change, can keep rules in separate file
  • Class attributes: Selection of stylesheets that only applies to certain classes

Git

  • Free, open source version control system
  • Allows collaboration on projects
  • Git Status: Checks what branch we are currently in
  • git checkout -b branch name: Creates new branch and switches too it
  • Don't forget to commit,save,and create new branches

JavaScript

  • Lightweight language that is used to provide logic and interactivity for websites
  • One of three core languages
  • Variable names Must be unique
  • Control Flow: The order the computer executes the code
  • Types of conditions: IF, Else If
  • Array: Groups of related data
  • Function: Set of instructions that tells the computer how to perform a task
